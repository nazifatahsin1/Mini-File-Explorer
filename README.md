# Finder Project

A GUI File Explorer built using Python and Tkinter.

## Features
- File navigation
- Search and sorting
- File operations (copy, paste, delete)
- Preview panel

## How to Run

1. Install Python 3.11
2. Clone the repository:
   git clone https://github.com/your-username/finder-project.git
3. Navigate to the project folder:
   cd finder-project
4. Run the application:
   python3.11 gui_explorer.py
